# Homebrew Tap for Neurox

Persistent memory for AI coding agents.

## Install

```bash
brew install joeldevz/neurox/neurox
```

Or first add the tap:

```bash
brew tap joeldevz/neurox
brew install neurox
```

## Update

```bash
brew upgrade neurox
```
